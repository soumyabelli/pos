🧾 POS System (Point of Sale Web Application)

A modern Point of Sale (POS) Web Application designed to streamline business operations such as billing, order management, inventory tracking, and customer handling.

This project is built using a full-stack architecture to provide a seamless and responsive experience for both administrators and users.

🚀 Features
🛒 Order Management
Add, update, and delete orders
Real-time cart system
💳 Payment Integration (Manual Verification)
QR-based payment (Paytm)
Upload transaction ID / screenshot
Admin verification system
📦 Inventory Management
Add, update, delete products
Stock tracking
👥 User Roles
Admin panel
Customer interface
📊 Dashboard
Sales overview
Order tracking
🔐 Authentication
Secure login/signup
Role-based access control
⚡ Responsive UI
Clean and modern interface
Works on desktop & mobile
